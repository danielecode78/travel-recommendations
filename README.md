# Travel Recommendation Website

A simple and responsive travel recommendation website built with HTML, CSS, and JavaScript. It showcases curated destinations including beaches, temples, and countries, with dynamic search functionality and a clean user interface.

## Features

- Responsive layout using Flexbox
- Dynamic search by name, type, or country
- Reset button to clear results
- Keyboard support (`Enter` triggers search)
- Semantic HTML and accessible form elements
- Clean design with image cards and hover effects

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Unsplash for royalty-free images
